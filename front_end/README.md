# React + Vite
# Frontend

This frontend provides a simple interface for the Aeroponic Tower Placement and Crop Recommendation System.

## Quick Start
1. Install dependencies:
	npm install
2. Run the app:
	npm run dev

## Structure
- src/: Source code
- components/: React components
- pages/: Pages
- services/: API service
- styles/: CSS
